正则表达式介绍

正则表达式据我的理解，就是用来匹配文字的，比如手机号、身份证号，以及项目中遇到过的一个v-has结合正则表达式进行权限判断等奇怪的操作。看到过一句话感觉挺贴切的：正则表达式类似于数学中的公司，有运算符优先级等说法，匹配的过程与套用公式过程类似。

四个重要方法

首先介绍三个重要的方法：search()、test()、replace()和match()

search()

search，字面上理解就是查找的意思，它会返回模式匹配到的第一个字符串的首字母的位置下标。

        console.log("Hello World!".search(/World/)); //6

test()

test 方法用于检测一个字符串是否匹配某个模式，如果字符串中含有匹配的文本，则返回 true，否则返回 false。

        const string = "The best things in life are free!"
        const patt1 = /ee/
        const patt2 = /hello/
        console.log(patt1.test(string)); //true
        console.log(patt2.test(string)); //false

replace()

replace用于将模式匹配到的字符串替换

        const string = "Hello World!"
        var str = string.replace(/World/i, "Vue");
        console.log(str); //Hello Vue!

match()

match 从字符串的开头位置匹配，从开头可以匹配到，就获得了一个正则对象。匹配不到返回None；

        var str = 'Hello World, my name is hxf';
        console.log(str.match(/name/)); //结果是一个数组，该数组一共有4项。
        /*
               第0项：匹配到字符串
        　　第1项：groups：undefined，这表示当前的正则表达式没使用分组
        　　第2项：index表示首次匹配上的子串的起始下标。
        　　第3项：input，表示源字符串
        　　第4项：length，表示匹配到的结果个数，由于这里不使用全局匹配，只找到首次匹配项就结束了，所以匹配结果只有1个，length也就是1。
        */
        // 如果在正则表达式末尾不使用全局匹配符g，在本例中是无法匹配到所有的数字。反过来说就是，如果想匹配所有的符合条件的子串，就必须在正则的末尾添加全局匹配符g
        // 加了全局修饰符g
        console.log(str.match(/name/g)); //['name']

修饰符

i    执行对大小写不敏感的匹配。
例如：/abc/i 可以匹配 abc、aBC、Abc
g    执行全局匹配（查找所有匹配而非在找到第一个匹配后停止）。
如果不带g，正则过程中字符串从左到右匹配，找到第一个符合条件的即匹配成功，返回一个正则对象。
如果带g，则字符串从左到右，找到每个符合条件的都记录下来，知道字符串结尾位置。
如上面的match()例子。
m    执行多行匹配。
s    特殊字符远点包含换行符

若存在换行\n并且有开始^或结束$符的情况下，和g一起使用实现全局匹配。
因为存在换行时默认会把换行符作为一个字符，任务匹配字符串是个单行，g只匹配第一行，添加m之后实现多行，每个换行符之后就是开始。

        var str = 'test\nbbs';
        var reg1 = /^b/g;
        console.log(str.match(reg1)); //null
        var reg2 = /^b/mg;
        console.log(str.match(reg2)); //['b']

语法

普通字符

[abc] 匹配[...]的所有字符

[^abc] 取反，除了[...]的其他字符

[A-Z] 区间字母A到Z

. 匹配除（\n换行符 \r 回车符）的任何单个字符

\s \S 匹配所有，\s所有空白符，包括换行 \S非空白符，不包括换行

\w 匹配字母、数字、下划线

        var str = 'Hello World, my name is hxf';
        console.log(str.match(/[abc]/g));//['a']
        console.log(str.match(/[^abcdefghijklmn]/g));// ['H', 'o', ' ', 'W', 'o', 'r', ',', ' ', 'y', ' ', ' ', 's', ' ', 'x']
        console.log(str.match(/[A-Z]/g));// ['H', 'W']
        console.log(str.match(/h.llo/ig));//['Hello']
        console.log(str.match(/\s/g));//[' ', ' ', ' ', ' ', ' ']
        console.log(str.match(/\w/g));//['H', 'e', 'l', 'l', 'o', 'W', 'o', 'r', 'l', 'd', 'm', 'y', 'n', 'a', 'm', 'e', 'i', 's', 'h', 'x', 'f']

特殊字符

$ 结尾位置

^ 开始位置

() 子表达式开始和结束

* 匹配零次或多次

+ 匹配一次或多次

? 匹配零次或一次

| 两项间的一个

        console.log(str.match(/me\?$/mg));//['me?']
        console.log(str.match(/^what/mg));//['what']
        console.log(str.match(/(^what).*(name\?$)/mg));//['what is your name?']   console.log(str.match(/su*\ssan/mg));//['suuuuu san']
        console.log(str.match(/s?an/g));//['san']
        console.log(str.match(/s(a|b)n/g));//['san']

限定符

{n} n为非负整数，匹配n次

{n,} n为非负整数，至少n次

{n,m} n为非负整数，n<=m,最少n次，最多m次

        var str = 'Hello World, my name is hxf.\nwhat is your name?\nsuuuuu san';
        console.log(str.match(/su{2}/g));//['suu']     console.log(str.match(/su{2,}/g));//['suuuuu']
        console.log(str.match(/su{2,4}/g));//['suuuu']

复杂（诡异）的栗子

        var date = new Date().toLocaleTimeString('chinese', { hour12: false })//当前时分秒，例如9:38:06
        console.log(date.match(/^([01]\d|2[01234]):([0-5]\d|60)/g)); //['09:38'],匹配当前时分
        console.log('有点意思abcd123456'.match(/[\u4e00-\u9fa5]/g)); //['有', '点', '意', '思']
       
        var str1 = '身高:178，体重：138，学号：318123456，密码:962495'
        // 打印出str1中所有的数字
        console.log(str1.match(/\d+/g)); //['178', '138', '318123456', '962495']
        // 打印出str1中所有的汉字
        console.log(str1.match(/[\u4e00-\u9fa5]+/g)); //['身高', '体重', '学号', '密码']
        var email1 = 'email@qq.vip.com'
        var email2 = '12525@qq.com'
        // 分为两个部分，第一个部分[\w]+用于匹配邮箱的名称，第二个部分[\w]+(.[A-Za-z]+){1,2}用于匹配域名。
        // \w表示匹配数字、字母、下划线，[\w]+表示匹配一个或多个数字、字母、下划线；[A-Za-z]用于匹配一个字母，
        // [A-Za-z]+用于匹配一个或多个字母，.[A-Za-z]+用于匹配以.（点号）开头并连接一个或多个字母，
        // {1,2}表示匹配表达式(.[A-Za-z]+)一次或者两次。
        var regexp = /[\w]+@[A-Za-z]+(\.[A-Za-z0-9]+){1,2}\b/g
        console.log(email1.match(regexp)); //['email@qq.vip.com']
        console.log(email2.match(regexp)); //['12525@qq.com']
